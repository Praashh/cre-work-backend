
# CreWork-Backend 


This is the backend of CreWork application.


## Local setup

- Copy .env.example to .env.

```bash
  cp .env.example .env
```

- Clone the repo.

```bash
   git clone https://github.com/<yourusername>/cre-work-backend.git
```

- Install dependencies

```bash
  npm install 
```

- Start the server.
```bash
  npm run dev
```
